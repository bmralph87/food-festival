## Tools used for this project
- JavaScript
- Node.js
- Express
- MongoDB
- Mongoose  
- Morgan
- Compression
- Heroku
- MongoDB Atlas

- Create a PWA for a food festival where users can view schedules and purchase tickets for the various events
